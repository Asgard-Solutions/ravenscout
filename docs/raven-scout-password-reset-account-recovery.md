# Raven Scout — Password Reset & Account Recovery

## Reset a Forgotten Password
1. Tap **Sign in** on the welcome screen → **Forgot password?**
2. Enter the email you registered with.
3. Check your inbox for a **6-digit code** from `support@asgardsolution.io`. It's valid for **15 minutes**.
4. Enter the code in the app, then set a new password.
5. Sign in with the new password.

If the code email doesn't arrive within ~2 minutes, check spam / promotions. Still nothing? Request a new code (old codes invalidate automatically).

## Change a Known Password
**Profile → Account → Change Password**. You'll be asked for your current password plus the new one (twice).

## Google Sign-In Users
If you signed up with **Continue with Google**, you do NOT have a password — you sign in via Google every time. To add an email/password fallback: Profile → Account → **Add password** → we email a verification code → set a password. Both sign-in methods then work.

## Sign in with Apple Users *(iOS)*
If you used **Sign in with Apple** to create your account, password sign-in is not used by default. If you also enabled Apple's **Hide My Email** relay, your real inbox is routed through Apple's `@privaterelay.appleid.com` address — manage that in your Apple ID settings. You can optionally add an email + password fallback under Profile → Account → **Add password**, the same way Google-only users do.

## Forgot Which Email You Used
Try each email you regularly use. Every code request against an unknown email silently no-ops — you won't get a wrong-address bounce. If you exhausted the options, email **support@asgardsolution.io** with the most likely two or three addresses; we can look up your account by paired Google subject id or App Store / Play Store purchase email.

## After Five Failed Code Attempts
Five wrong-code entries on the same code burn that code. Request a new one and try again — there is no separate cooldown timer.

## Delete Your Account
**Profile → Account → Delete account** (bottom of the page). This is permanent — your saved hunts, Pro cloud imagery, and subscription entitlement are removed. Active App Store / Play Store subscriptions must be cancelled separately in the store.
