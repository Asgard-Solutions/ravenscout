# Raven Scout — Password Reset & Account Recovery

## Reset a Forgotten Password
1. Tap **Sign in** on the welcome screen → **Forgot password?**
2. Enter the email you registered with.
3. Check your inbox for a **6-digit code** from `support@asgardsolution.io`. It's valid for 10 minutes.
4. Enter the code in the app, then set a new password.
5. Sign in with the new password.

If the code email doesn't arrive within ~2 minutes, check spam / promotions. Still nothing? Request a new code (old codes invalidate automatically).

## Change a Known Password
**Profile → Account → Change Password**. You'll be asked for your current password plus the new one (twice).

## Google Sign-In Users
If you signed up with **Continue with Google**, you do NOT have a password — you sign in via Google every time. To add a password for fallback: Profile → Account → **Add password** → we email a verification code → set a password. Both sign-in methods now work.

## Forgot Which Email You Used
Try each email you regularly use. Every code request against an unknown email silently no-ops — you won't get a wrong-address bounce. If you exhausted the options, email **support@asgardsolution.io** with the most likely two or three addresses; we can look up your account by paired Google subject id or App Store / Play Store purchase email.

## Account Locked After Too Many Code Attempts
Five failed code attempts in 10 minutes triggers a 15-minute cooldown on password-reset attempts for that email. Wait it out and try again, or contact support for a manual reset.

## Delete Your Account
**Profile → Account → Delete account** (bottom of the page). This is permanent — all saved hunts, Pro cloud imagery, and subscription entitlements are removed within 30 days per our privacy policy. Active App Store / Play Store subscriptions must be cancelled separately in the store.
